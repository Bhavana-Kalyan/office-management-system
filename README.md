# Office Management System

A console-based Office Management System built using Python OOP and SQLite3.

## Features
- Role-based login (Admin and Employee)
- Add, View, Delete Employees
- Update Salary and Bonus
- Leave Management System
- Change Password feature

## Technologies Used
- Python 3
- SQLite3
- OOP Concepts

## How to Run
1. Run database setup: `python database.py`
2. Run application: `python app.py`

## Project Structure
- `employee.py` - Employee class with OOP
- `database.py` - Database and table setup
- `app.py` - Main application with all features
